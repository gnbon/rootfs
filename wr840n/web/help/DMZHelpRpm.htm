DMZ/스마트 DMZ 도움말

DMZ 호스트 기능은 DMZ 호스트로 지정된 장비로 모든 네트워크 데이터를 전달함으로써 공유기 내부에 있는 장비(PC 등)가 게임, 화상회의, 서버 운영 등, 특별한 기능이 원활하게 동작할 수 있게 지원해 줍니다. DMZ 호스트 IP주소는 공유기의 DHCP 기능에 의해서 IP가 변경 될 수 있으므로, IP주소가 변경되지 않게 공유기에서 사용하지 않는 IP주소로 고정 IP 방식으로 지정하여 주십시오.

스마트 DMZ 기능은 공유기 내부에 있는 특정 PC에 공유기의 인터넷 주소와 동일한 주소를 할당합니다. 해당 PC는 공유기가 없는 환경과 동일하게 동작하므로 공유기 환경에서 실행되지 않는 게임 등의 사용을 위해서 이 기능을 사용할 수 있습니다.

DMZ 설정 방법:

  1. DMZ 버튼을 선택 하십시오.
  2. DMZ 기능 사용을 위한 PC의 IP주소를 DMZ 호스트 IP 주소 영역에 입력하십시오.
  3. 설정 완료 후 저장 버튼을 클릭하십시오.

스마트 DMZ 설정 방법:

  1. 스마트 DMZ 버튼을 선택 하십시오.
  2. 스마트 DMZ 기능을 사용할 PC의 MAC 주소 를 입력하십시오.
  3. PC의 인터넷 주소 갱신을 위한 IP 갱신 주기를 입력하십시오.
  4. 스마트 DMZ PC의 DHCP 임대기간은 60초로 고정됩니다.
