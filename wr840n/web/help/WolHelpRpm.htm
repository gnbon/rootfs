WOL 설정 도움말

웨이크 온 랜(Wake-on-LAN, WOL)은 네트워크 메시지를 보냄으로써 컴퓨터의 전원을 켜거나 절전 모드에서 깨어나게 하는 이더넷 컴퓨터 네트워킹 표준입니다.

참고: 이 기능은 사용자의 파워, 메인보드, 네트워크 어댑터가 WOL프로토콜을 지원해야 사용이 가능합니다.

WOL 설정:

  • MAC주소 입력란에 해당 PC의 MAC주소를 입력하세요.
  • PC이름 입력란에 해당 PC의 이름을 입력하세요
  • 추가 버튼을 누르면 해당 리스트의 끝에 새로운 아이템이 첨부됩니다.
  • MAC주소 복제를 누르면 해당 MAC주소가 좌측의 텍스트 박스에 복사됩니다.
  • PC검색 버튼을 클릭하여 LAN상의 기존 모든 PC의 MAC과 IP주소를 검색합니다.

WOL 리스트:

  • 켜기 버튼을 누르면 PC의 전원이 켜집니다.
  • 삭제버튼을 누르면 해당 아이템이 리스트에서 삭제됩니다.

WOL 검색 리스트:

  • 로드 버튼을 누르면 MAC주소가 MAC주소 입력란에 복사되고, 이전 페이지로 돌아갑니다.
  • 이전 버튼을 누르면 이전 페이지로 바로 이동합니다.
